# 系统架构

## 1. 总体架构

IndexPlatform 分为以下层：

1. 数据源层
2. 数据标准化层
3. 本地存储层
4. 指数注册层
5. 策略层
6. 回测层
7. 绩效指标层
8. CLI 层
9. Dashboard 层
10. 运行监控层

CLI 和 Dashboard 必须复用同一套核心模块。
