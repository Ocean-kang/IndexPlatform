# Codex 任务清单

本文件用于指导 Codex 按阶段开发 IndexPlatform。

核心原则：

* 一次只做一个任务。
* 每个任务都必须可测试、可运行、可提交。
* 不要提前实现后续任务。
* CLI 和 Dashboard 必须复用同一套核心逻辑。
* 第一阶段只做研究和回测，不做实盘交易。
* 第一阶段只做日频数据，不做分钟级数据。
* 第一阶段不做机器学习预测，等核心回测闭环稳定后再做。

---
