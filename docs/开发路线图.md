# 开发路线图

## Phase 0：项目初始化

目标：

- 创建 Python 包结构
- 添加 pyproject.toml
- 添加 pytest
- 添加基础 CLI
- 添加基础文档

验收标准：

- pytest 可以运行
- python -m index_platform.cli.main --help 可以运行
- 包可以正常 import

## Phase 1：指数注册表

目标：

- 添加 configs/indices.yaml
- 加载指数元数据
- 校验指数字段

验收标准：

- 可以列出支持的指数
- 可以按市场过滤指数
- 测试通过

## Phase 2：本地数据存储

目标：

- 支持 CSV 导入
- 校验日频价格 schema
- 保存标准化数据到 Parquet
- 按 symbol 和日期范围读取价格数据

验收标准：

- idx data import 可用
- idx data status 可用
- 数据读写测试通过

## Phase 3：基础回测引擎

目标：

- 实现买入持有回测
- 生成净值序列
- 保存回测结果

验收标准：

- 回测能生成 NAV
- 回测有 run_id
- 简单价格序列的结果可验证

## Phase 4：绩效指标

目标：

- 实现基础绩效指标

验收标准：

- 累计收益
- 年化收益
- 年化波动率
- 最大回撤
- 夏普比率
- Calmar 比率

## Phase 5：策略模块

目标：

- 均线择时
- 动量轮动

验收标准：

- 策略能输出目标权重
- 信号和权重有测试

## Phase 6：CLI

目标：

- 形成稳定 CLI 命令

验收标准：

- idx list-indices
- idx data import
- idx data status
- idx backtest run
- idx report show

## Phase 7：Dashboard

目标：

- 创建 Streamlit 可视化界面

验收标准：

- 可以查看数据状态
- 可以查看指数走势图
- 可以查看回测结果

## Phase 8：运行监控

目标：

- 记录任务状态

验收标准：

- 回测任务写入 running / success / failed 状态
- CLI 和 dashboard 能读取任务状态
