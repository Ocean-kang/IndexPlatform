# IndexPlatform 产品说明

## 1. 产品定位

IndexPlatform 是一个本地优先的指数投资研究平台。

它用于帮助用户：

- 收集指数数据
- 标准化指数数据
- 展示指数历史表现
- 回测指数投资策略
- 比较策略和基准指数
- 可视化策略结果
- 监控策略运行状态

第一阶段是研究和回测平台，不是实盘交易系统。

## 2. 目标市场

平台未来支持：

- A 股指数
- 港股 / H 股指数
- 美股指数

第一版优先支持：

- A 股主要指数
- 本地 CSV 数据
- 本地 Parquet 数据存储

## 3. 第一版范围

第一版支持：

- 指数注册表
- 日频指数价格数据
- 本地 CSV 导入
- 本地 Parquet 存储
- 基础数据校验
- 买入持有回测
- 均线择时策略
- 动量轮动策略
- 基础绩效指标
- CLI 命令
- 简单 Streamlit 可视化界面

## 4. 第一版不做什么

第一版不实现：

- 实盘交易
- 券商接口
- 分钟级回测
- 期权
- 期货
- 融资融券
- 杠杆
- 卖空
- 复杂组合优化
- 机器学习预测
- 云端部署
- 用户账号系统

## 5. 核心用户流程
